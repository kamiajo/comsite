Требования для поступления

DEADLINE

  • Документы для поступления в магистратуру принимаются до 30.09.17 в Международном университете Алатоо,
    по адресу г. Бишкек, мкр.Тунгуч, ул.Анкара, 1/8
    Телефоны для справок: 0312 54-92-38, 0312 63-14-25

    • Note for international students: На заметку иностранным гражданам Перевод паспорта, диплома и его вкладыша на кыргызский либо на русский языки (перевод должен быть подтвержден у сертифицированного нотариуса или переводческих фирм).

Требования по специальности:

    Диплом бакалавра или специалиста по информатике с успеваемостью 75% за последние два года обучения. Принимается также диплом бакалавра или специалиста по техническому направлению с успеваемостью 85% по предметам, связаные с информатикой, за последние два года обучения. Данные компетенции должны быть засвидетельствованы документально. Профессиональная деятельность в сфере программирования может быть признана в качестве выполнения критерия для поступления. При необходимости студент должен послушать курсы по углубленному программированию и по разработке программного обеспечения в программе бакалавриата КГФПИ или Университета Алатоо. Эти обязательные курсы читаются не вечером и не на выходных. Для возможных курсов взимается плата дополнительно к основной сумме платы за обучение.

Требования по иностранным языкам:

  • Английский язык B2 или

  • Английский язык B1 или немецкий язык B2

  • Уровень владения английским языком определяется преподавателями Университета Алатоо.

  • Уровень владения немецким языком определяется преподавателями КГФПИ КГУСТА

  • Заявление

  • Подлинник диплома бакалавра (специалиста) с приложением выписки из зачетной ведомости

  • 2 фотографии 3х4

  • Копия паспорта (оригинал для просмотра)

  • Договор
